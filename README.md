# C_Avoid_the_Ball_PCandAndroid
#4. 공을 피해 오래 살아남는 게임(PC, 모바일)

## 1. 개발 계기

```
2019년 3학년 2학기 2D게임(PC, Android) 게임 개발
게임프로그래밍 과목을 수강, 학교 공모전을 참가하기 위해 1인 개발.
2D와 3D는 차이가 크다. 
내가 개발한 게임은 3D 게임이 많았으며 2D게임을 만들고 싶어져,
레트로 감성이 묻어나는 2D게임으로 개발하였다.
```
